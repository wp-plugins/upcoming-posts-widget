A widget that displays a list of scheduled posts.

== Description ==

Upcoming Posts Widget acts very much like the recent posts widget, but displays
scheduled posts instead of published ones.  Requires WordPress 2.8+

You may also be interested in WordPress tips and tricks at <a href="http://wpinformer.com">WordPress Informer</a>
